# Glossar
* Server: Raspberry Pi
    * Contains playlist(s) and it's containing files
    * Managing clients and sessions
* Client
    * Session: Client which is playing and controlling media
    * Master: Session-Holder
    * Slave: Client which listens to a Master's session
